📊 Project Overview

This project maps educational institutions in Rwanda's conflict-affected regions. The data combines OpenStreetMap school locations with vulnerability analysis to support humanitarian and development initiatives.

🌐 Primary Data Sources

1. OpenStreetMap (OSM) - Schools Database

OpenStreetMap Global Database

Primary Source Free & Open

Description: Raw school location data extracted from OpenStreetMap's global database. OSM is a collaborative, open-source project where volunteers contribute geographical data worldwide.

Coverage: 1,602 school locations across Rwanda with attributes including:

  • School name and official designation
  • Precise geographic coordinates (latitude/longitude)
  • Street address and city information
  • Last verification date
  • OSM object type and ID

Last Updated: April 2023

→ Visit OpenStreetMap

2. Conflict-Affected Regions Classification

Rwanda Geographic Conflict Impact Analysis

Analysis Custom

Description: Classification of schools based on their proximity to historically conflict-affected areas and current vulnerability assessments.

Methodology: Spatial analysis using conflict-affected region boundaries and humanitarian impact data.

Reference Points:

  • Kigali (Capital) - 30.0587°E, -1.9441°S
  • Huye (Butare) - 29.7479°E, -2.5956°S
  • Rubavu (Gisenyi) - 29.2549°E, -1.7030°S

🔍 Data Collection Methods

OpenStreetMap Query Process

Query Method: Overpass API (OSM Query Language)

Filters Applied:

  • amenity = "school"
  • Geographic bounds: Rwanda administrative boundaries
  • Data validation: Coordinates within country bounds

Processing Steps:

  1. Export raw OSM data via Overpass API
  2. Convert to GeoJSON format
  3. Validate geographic coordinates
  4. Cluster analysis for spatial patterns
  5. Generate both clustered and individual views
